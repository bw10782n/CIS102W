HTML CHEAT SHEET

BASIC HTML ELEMENTS

< DOCTYPE! >
Document Type Definition, specifies the language and version of document
< html >
Root of HTML document, defines that it is an HTML
< head >
Defines information about the document
< body >
The body of the document
< h1> - < h6 >
Header tags
< p >
Paragraph tags
align = "center", "left", "right"

LIST TAGS

< ol >
Ordered list
< ul >
Unordered list
< li >
List item

LINK AND IMAGE/MEDIA TAGS

< a >
Defines a hyperlink or a link to another document
href, hreflang, media, ping, rel, target, type
< img >
Defines an image
alt (to add description to image), src, height, width
< audio >
Adds audio content to web page
autoplay, controls
< address >
To present an address in web page
< video >
Embeds a media player for video playback in the document. Can be used for audio as well.
autoplay, controls, loop, height, width, src

STYLING AND TEXT FORMATTING TAGS

< b >
Makes the text bold
< strong >
Conveys an extra importance in text
< em >
Conveys emphasis in text
< big>
Big text
< i >
Italic text
< small >
Small text
< sub >
Subscripted text
< sup >
Superscripted text
< ins >
Inserted text
< del >
Deleted text
< - - Text - - > - -
Defines a comment
< abbr >
Creates an abbreviation
< br >
To indicate a line break
< section > & < article >
Defines a section or article, can be nested inside one another
Section tags are used to represent a standalone section.
Article tags are used to represent a self contained composition in a document, page, application or site which is intended to be distributed or reused independently
< div >
Creates a block section in document where a line break is placed before and after it
< span >
In line section in document

TABLE TAGS

< table >
Defines a table
align, border, height, width, bgcolor
< tr >
Defines table row
< th >
Defines table header
< td >
Defines table row data
align, height, width, border, bgcolor

FORM TAGS

< form >
Creates a form input to collect information from user
< input >
Defines the input field type
type= text, password, name, color, date, file, image, number, url, etc.

  • input type= button CREATES A BUTTON
  • < input id="mysearch" name="searchitem" type="search" /> CREATES A SEARCH BOX
< label for >
To create a label for each field being asked
COULD BE ANYTHING ex: name, email, message
placeholder= *for what needs to be typed in field box*

HTML5 TAGS

< header >
For introductory content such as a logo, search form or author name. Goes in < body > tag
< footer >
Contains information about the author of section, copyright data, or additional links.
< nav >
Represents a section whose purpose is to provide navigation links within the current document or to others. Ex: Menus, table of contents & indexes
< aside >
Represents a portion of document whose content is indirectly related to the main content.
< progress >
To show progress on webpage
max, value
< datalist >
Defined a dropdown list of pre-defined items
